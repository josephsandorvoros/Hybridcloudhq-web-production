Walk the Fellowship route

This version is built specifically for the web and focuses on a clean manual tracker. Progress is cached locally, the route updates instantly, and you can add your own high-resolution map image in the assets folder.

  • Default stride length is 0.76m and can be adjusted from 0.5m to 1.2m.
  • Distance can be shown in kilometers, miles, or meters.
  • Drop a file named middle-earth-map.webp, .png, .jpg, or .jpeg into assets to use your own map.
